# JS Board

Vue.js와 Node.js 서버를 연결하여 설계한 API를 통해 서버에 요청을 하면  
MongoDB에 저장된 데이터를 CRUD 할 수 있는 게시판 웹사이트입니다.

## 구현 기능

<ul>
  <li>게시글 추가</li>
  <li>게시글 읽기</li>
  <li>게시글 수정</li>
  <li>게시글 삭제</li>
</ul>

## 기술 스택

<ul>
  <li>Vue.js</li>
  <li>Bootstrap</li>
  <li>Node.js</li>
  <li>MongoDB</li>
</ul>
